## Visao Neurodivergente: Como Esta Ferramenta Apoia o Cerebro TDAH

---

## 1. O Cerebro TDAH: Entendendo Nosso Publico

### O Que Torna o Cerebro TDAH Diferente?

```
CEREBRO NEUROTYPICO          CEREBRO TDAH

    Foco Estavel             Foco Intenso ou Zero
    Energia Linear           Energia em Ondas
    Tempo Previsivel         "Cegueira Temporal"
    Motivacao Constante      Movido por Interesse
```

### Os 5 Desafios Centrais do TDAH

| Desafio | Como Se Manifesta | Impacto no Dia a Dia |
|---------|-------------------|---------------------|
| **Funcao Executiva** | Dificuldade em planejar, priorizar e iniciar | Procrastinacao, tarefas incompletas |
| **Regulacao Emocional** | Emocoes intensas, frustracao rapida | Abandono de projetos, burnout |
| **Memoria de Trabalho** | Esquece tarefas, perde o fio | Coisas "somem" da mente |
| **Percepcao de Tempo** | Tempo e abstrato, prazos sao surpresa | Atrasos cronicos, correria |
| **Paralisia por Analise** | Muitas opcoes = travamento | Nao consegue comecar |

### O Que FUNCIONA Para o Cerebro TDAH

- **Dopamina**: Novidade, recompensa imediata, gamificacao
- **Externalizacao**: Tirar da cabeca, colocar no papel/app
- **Reducao de Atrito**: Menos cliques, menos decisoes
- **Urgencia Artificial**: Timers, prazos visiveis
- **Validacao**: Celebrar pequenas vitorias

## 2. Mapa de Recursos: Cada Funcionalidade vs. Necessidade TDAH

### 2.1 Dashboard Principal

```
+------------------------------------------+
|  BOM DIA, [NOME]!                        |
|  "Voce tem 3 tarefas para hoje"          |
|                                          |
|  [ENERGIA: 7/10]  [HUMOR: Focado]        |
|                                          |
|  TOP 1 DE HOJE:                          |
|  [ ] Revisar relatorio - Alta Prior.     |
|                                          |
|  PROXIMAS:                               |
|  [ ] Reuniao 14h                         |
|  [ ] Responder emails                    |
+------------------------------------------+
```

**Por que funciona para TDAH:**
- Saudacao personalizada = conexao emocional
- Numero limitado de tarefas = sem overwhelm
- TOP 1 = decisao ja tomada, so executar
- Indicadores de energia/humor = autoconhecimento

**Oportunidades de melhoria:**
- [ ] Mostrar QUANTO TEMPO falta para proxima tarefa
- [ ] Cor/visual que muda conforme energia
- [ ] Frase motivacional baseada no estado atual

### 2.2 Brain Dump (Captura Rapida)

```
+------------------------------------------+
|  BRAIN DUMP                              |
|  "Tire da cabeca, coloque aqui"          |
|                                          |
|  [________________________] [+]          |
|                                          |
|  INBOX (5 itens):                        |
|  - Ligar pro dentista                    |
|  - Ideia: app de receitas                |
|  - Comprar presente mae                  |
|  - Revisar contrato                      |
|  - Pesquisar curso                       |
+------------------------------------------+
```

**Por que funciona para TDAH:**
- Captura IMEDIATA = nao perde o pensamento
- Sem categoria obrigatoria = zero atrito
- Inbox = pode processar depois com calma

**Oportunidades de melhoria:**
- [ ] Captura por VOZ (falar e mais rapido que digitar)
- [ ] Botao flutuante em TODAS as paginas
- [ ] Timer opcional: "Processar inbox em 5 min"

```

|                                          |

|  [ ] Arrumar gaveta                      |

```

**Por que funciona para TDAH:**
- Cores = processamento visual rapido
- Prioridades claras = menos decisao
- Check = dopamina imediata

**Oportunidades de melhoria:**
- [ ] Estimativa de tempo em cada tarefa
- [ ] "Quanto esforco?" (Facil/Medio/Dificil)
- [ ] Sugestao automatica: "Sua energia esta baixa, que tal uma tarefa facil?"
- [ ] Celebracao visual ao completar (confete, som, frase)

### 2.4 Modo Crise

```
+------------------------------------------+
|  MODO CRISE ATIVADO                      |
|  "Esta tudo bem. Vamos devagar."         |
|                                          |
|  RESPIRE:                                |
|       ○ ○ ○ ○ (animacao de respiracao)   |
|                                          |
|  AGORA, ESCOLHA APENAS UMA:              |
|                                          |
|  [Beber agua]  [Levantar]  [5min pausa]  |
|                                          |
|  Todas as outras tarefas foram           |
|  pausadas. Foque apenas em voce.         |
+------------------------------------------+
```

**Por que funciona para TDAH:**
- Reconhece o estado emocional
- Reduz opcoes ao MINIMO
- Linguagem acolhedora, nao critica
- Acoes fisicas simples (movimento ajuda)

**Oportunidades de melhoria:**
- [ ] Detectar crise automaticamente (palavras-chave no chat)
- [ ] Opcao de "Falar com alguem" (contato de emergencia)
- [ ] Musica/som ambiente calmante
- [ ] Historico de crises para autoconhecimento

### 2.5 Assistente IA (Chat)

```
+------------------------------------------+
|  ASSISTENTE IA                           |
|                                          |
|  Voce: "Estou travado, nao sei o que     |
|         fazer primeiro"                  |
|                                          |
|  IA: "Entendo! Olhei suas tarefas.       |

|       sugiro comecar por:                |
|                                          |
|       1. Revisar relatorio (30min)       |
|          - E importante e voce           |
|            esta com energia              |
|                                          |
|       Quer que eu crie um timer?"        |
|                                          |
|  [Sim, criar timer]  [Ver outras opcoes] |
+------------------------------------------+
```

**Por que funciona para TDAH:**
- Externaliza a decisao (IA decide por voce)
- Considera CONTEXTO (energia, hora, tarefas)
- Oferece ACAO imediata (timer)
- Tom de parceiro, nao de chefe

**Oportunidades de melhoria:**
- [ ] Memoria de longo prazo (lembrar padroes)
- [ ] "Voce sempre trava as 15h, que tal uma pausa?"
- [ ] Integracao com calendario real
- [ ] Comandos de voz

## 3. Jornada do Usuario TDAH

### Cenario: Segunda-Feira de Manha

```
07:30  ACORDA
       └─> App mostra: "Bom dia! Check-in rapido?"

07:35  CHECK-IN
       └─> Energia: 6/10, Humor: 😐, Dormiu: Mais ou menos
       └─> App sugere: "Dia de energia media.
           Comece com tarefas leves."

08:00  DASHBOARD
       └─> TOP 1: "Responder email importante"
       └─> App: "Essa tarefa leva ~15min. Comecar agora?"

08:15  TAREFA CONCLUIDA
       └─> Celebracao! "Otimo! 1 de 4 concluida"
       └─> Proxima sugestao automatica

10:30  TRAVOU
       └─> Chat: "Estou sem foco, nao consigo continuar"
       └─> IA: "Que tal uma pausa de 10min?
           Depois, uma tarefa facil para retomar."

12:00  ALMOCO
       └─> App silencia notificacoes
       └─> Lembrete gentil as 13:00

15:00  CRISE
       └─> Usuario: "Nao aguento mais"
       └─> Modo Crise ativa automaticamente
       └─> Respiracao guiada + opcoes simples

17:00  FIM DO DIA
       └─> Review: "Voce completou 3 de 4 tarefas!"
       └─> Celebracao + planejamento amanha
```

## 4. Principios de Design para TDAH

### O Que FAZER

| Principio | Exemplo na Pratica |
|-----------|-------------------|
| **Menos e mais** | Maximo 3-5 tarefas visiveis por vez |
| **Decisoes pre-tomadas** | "TOP 1" ja escolhido pelo sistema |
| **Feedback imediato** | Confete ao completar tarefa |
| **Flexibilidade** | Remarcar sem culpa |
| **Tom acolhedor** | "Tudo bem, amanha e outro dia" |
| **Externalizacao** | Tudo visivel, nada so na cabeca |
| **Gamificacao leve** | Streaks, pontos, conquistas |

### O Que NAO FAZER

| Anti-Padrao | Por Que Prejudica |
|-------------|-------------------|
| Listas infinitas | Paralisia, overwhelm |
| Muitas cores/icones | Distrai, confunde |
| Notificacoes excessivas | Ansiedade, ignora tudo |
| Tom critico | "Voce nao fez X" = culpa |
| Complexidade | Muitos cliques = abandono |
| Rigidez | Nao adapta ao dia |

## 5. Gaps e Oportunidades

### Funcionalidades que FALTAM

1. **Timer Pomodoro Integrado**
   - TDAH funciona bem com sprints curtos
   - 25min foco + 5min pausa
   - Visual de progresso

2. **Body Doubling Virtual**
   - Trabalhar "junto" com outros
   - Sessoes de foco em grupo
   - Accountability partner

3. **Recompensas e Gamificacao**
   - Pontos por tarefas
   - Streaks de dias produtivos
   - Conquistas desbloqueaveis

4. **Integracao com Wearables**
   - Dados de sono do smartwatch
   - Variabilidade cardiaca = estresse
   - Lembretes no pulso

5. **Modo Hiperfoco**
   - Quando esta "na zona"
   - Bloqueia distraçoes
   - Lembra de pausar/comer

### Melhorias de UX

1. **Onboarding Personalizado**
   - "Qual seu maior desafio?"
   - Configurar app para SEU cerebro

2. **Temas Visuais**
   - Modo escuro
   - Cores personalizaveis
   - Reducao de estimulos

3. **Acessibilidade**
   - Tamanho de fonte
   - Contraste alto
   - Leitura em voz alta

## 6. Metricas de Sucesso

### O Que Medir

| Metrica | O Que Indica | Meta |
|---------|-------------|------|
| Tarefas completadas/dia | Produtividade | Aumento 20% |
| Tempo ate primeira tarefa | Reducao de procrastinacao | < 15min |
| Uso do Modo Crise | Regulacao emocional | Reducao 30% |
| Check-ins realizados | Engajamento | > 80% dias |
| Retencao 30 dias | Valor percebido | > 60% |

### Feedback Qualitativo

- "Me sinto menos sobrecarregado"
- "Finalmente consigo comecar o dia"
- "A IA me entende"
- "Nao me sinto julgado"

## Anexo: Glossario TDAH

| Termo | Significado |
|-------|-------------|
| **Funcao Executiva** | Habilidades de planejamento, organizacao, foco |
| **Hiperfoco** | Estado de concentracao intensa (pode ser bom ou ruim) |
| **Paralisia por Analise** | Travar por ter muitas opcoes |
| **Cegueira Temporal** | Dificuldade em perceber passagem do tempo |
| **Body Doubling** | Trabalhar na presenca de outros para manter foco |
| **Dopamina** | Neurotransmissor de recompensa, deficiente no TDAH |
| **Overwhelm** | Sobrecarga, sensacao de "demais" |
| **Burnout** | Esgotamento por excesso de esforco |
