## Perguntas a Responder Antes de Criar Stories

1. **Central de Descoberta**
   - Qual tecnologia para o servidor central? (Supabase Realtime? Socket.io? SignalR?)
   - Como funciona a descoberta? (QR code? ID unico? Email?)

2. **Comunicacao P2P**
   - WebRTC direto ou via TURN server?
   - Fallback para servidor quando P2P falhar?

3. **Chat de Equipe**
   - Mensagens persistidas no servidor ou apenas P2P?
   - Historico de mensagens? Por quanto tempo?

4. **Dashboard do Gestor**
   - Quais metricas sao anonimizadas?
   - Gestor ve dados individuais ou apenas agregados?

5. **Modelo de Dados**
   - Estrutura de Team, TeamMember, TeamInvite?
   - Como relacionar com Users existentes?
