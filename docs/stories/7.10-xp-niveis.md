# Story 7.10: Sistema de XP e Níveis

## Status
Draft

## Story

**As a** usuário,
**I want** ganhar XP e subir de nível ao completar ações,
**so that** eu tenha uma sensação de progresso e motivação.

## Acceptance Criteria

1. Ganhar XP por ações (tarefas, check-ins, rotinas)
2. Sistema de níveis (1-50+)
3. Barra de progresso visual
4. Notificação ao subir de nível
5. XP nunca é perdido (só ganha)
6. Configurável (pode esconder)

## Tasks / Subtasks

- [ ] **Sistema de XP**
  - [ ] Tarefa completada: 10-30 XP (por prioridade)
  - [ ] Check-in: 5 XP
  - [ ] Rotina completa: 20 XP
  - [ ] Dia sem modo crise: 15 XP
  - [ ] Streak de 7 dias: 50 XP bonus

- [ ] **Sistema de Níveis**
  - [ ] Fórmula: XP necessário = nivel * 100
  - [ ] Nível 1: 0-100 XP
  - [ ] Nível 2: 100-200 XP
  - [ ] etc.

- [ ] **UI**
  - [ ] Badge de nível no perfil
  - [ ] Barra de progresso no dashboard
  - [ ] Animação de level up
  - [ ] Toggle para esconder
