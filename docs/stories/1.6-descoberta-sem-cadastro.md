# Story 1.6: Fluxo de Descoberta sem Cadastro

## Status
Draft

## Story

**As a** usuário curioso,
**I want** fazer a descoberta sem precisar criar conta,
**so that** receba valor imediato sem compromisso.

## Acceptance Criteria

1. Descoberta pode ser iniciada sem login
2. Resultado é exibido normalmente
3. Ao tentar salvar, usuário é convidado a criar conta
4. Dados do questionário são preservados durante criação de conta
5. Experiência fluida sem perda de dados

### Session ID Management
```typescript
// utils/session.ts
import AsyncStorage from '@react-native-async-storage/async-storage';
import 'react-native-get-random-values';
import { v4 as uuidv4 } from 'uuid';

const SESSION_KEY = '@nciaflux/anonymous_session';

export async function getOrCreateSessionId(): Promise<string> {
  let sessionId = await AsyncStorage.getItem(SESSION_KEY);
  if (!sessionId) {
    sessionId = uuidv4();
    await AsyncStorage.setItem(SESSION_KEY, sessionId);
  }
  return sessionId;
}

export async function clearSessionId(): Promise<void> {
  await AsyncStorage.removeItem(SESSION_KEY);
}
```

### Edge Function de Migração
```typescript
// supabase/functions/migrate-anonymous-profile/index.ts
serve(async (req) => {
  const { sessionId, userId } = await req.json();

  const supabase = createClient(/* ... */);

  // Buscar profile anônimo
  const { data: profile } = await supabase
    .from('profiles')
    .select('*')
    .eq('session_id', sessionId)
    .single();

  if (!profile) {
    return new Response(JSON.stringify({ error: 'Profile not found' }), { status: 404 });
  }

  // Migrar para usuário
  const { error } = await supabase
    .from('profiles')
    .update({ user_id: userId, session_id: null })
    .eq('id', profile.id);

  if (error) throw error;

  return new Response(JSON.stringify({ success: true, profile }));
});
```

### Cleanup de Sessões Órfãs
```sql
-- Rodar periodicamente (pg_cron)
DELETE FROM profiles
WHERE session_id IS NOT NULL
  AND user_id IS NULL
  AND created_at < NOW() - INTERVAL '7 days';
```
