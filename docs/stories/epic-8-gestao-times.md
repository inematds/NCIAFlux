**Definition of Done do Epic:**
- [ ] Gestor consegue criar time
- [ ] Convites por código/link funcionam
- [ ] Roles diferentes têm permissões diferentes
- [ ] Dashboard mostra métricas do time (anonimizadas)
- [ ] Gestor pode liberar/bloquear features para membros
- [ ] Relatórios de time exportáveis
- [ ] Desafios semanais de time funcionam
- [ ] Rankings (opt-in) funcionam sem pressão
- [ ] Conquistas de time desbloqueiam juntas
- [ ] Testes de integração passando

**Regras de Privacidade:**
- Gestor NUNCA vê dados individuais de humor/energia
- Métricas são sempre agregadas e anonimizadas
- Membro pode optar por não participar de rankings
- Dados pessoais continuam privados
