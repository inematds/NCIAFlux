# Story 7.11: Conquistas e Streaks Gentis

## Status
Draft

## Story

**As a** usuário com TDAH,
**I want** desbloquear conquistas e manter streaks sem punição,
**so that** eu tenha motivação sem ansiedade de perder progresso.

## Acceptance Criteria

1. Sistema de conquistas (badges)
2. Streaks que pausam em vez de zerar
3. Notificação gentil ao desbloquear
4. Galeria de conquistas no perfil
5. Conquistas nunca são perdidas
6. Mensagens de encorajamento, não cobrança

## Tasks / Subtasks

- [ ] **Conquistas**
  - [ ] ~20 conquistas iniciais
  - [ ] Categorias: Início, Consistência, Bem-estar, Social
  - [ ] Ícones e descrições

- [ ] **Streaks Gentis**
  - [ ] Contador de dias ativos
  - [ ] Pausa automática (não zera)
  - [ ] Mensagem: "Descansou 2 dias. Sem problemas!"
  - [ ] Recorde pessoal salvo

- [ ] **UI**
  - [ ] Toast ao desbloquear conquista
  - [ ] Página de conquistas
  - [ ] Progresso para próximas
