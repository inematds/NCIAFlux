# Story 7.6: Motor de Adaptação

## Status
Draft

## Story

**As a** usuário com TDAH,
**I want** que o app se adapte automaticamente baseado nos meus padrões,
**so that** a experiência seja otimizada para mim sem esforço.

## Acceptance Criteria

1. Calcular score de estabilidade do usuário (0-100)
2. Adaptar número de tarefas visíveis baseado em energia
3. Sugerir pausas em horários de baixa energia
4. Ativar "modo gentil" automaticamente quando detectar overwhelm
5. Ajustar notificações por horário/contexto
6. Permitir override manual das adaptações

## Tasks / Subtasks

- [ ] **Algoritmo de Estabilidade**
  - [ ] Pesos: check-in (25%), humor (20%), tarefas (20%), crise (20%), rotinas (15%)
  - [ ] Calcular diariamente
  - [ ] Histórico de 30 dias

- [ ] **Adaptações de UI**
  - [ ] Esconder tarefas quando energia < 3
  - [ ] Mostrar só Top 1 quando overwhelm detectado
  - [ ] Modo gentil automático

- [ ] **Sugestões Contextuais**
  - [ ] "Sua energia costuma cair agora, que tal uma pausa?"
  - [ ] "Horário de pico! Boa hora para tarefa importante"

- [ ] **Configurações**
  - [ ] Toggle adaptação automática ON/OFF
  - [ ] Override manual por sessão
  - [ ] Log de adaptações aplicadas
