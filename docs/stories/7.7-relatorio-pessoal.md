# Story 7.7: Relatório Pessoal

## Status
Draft

## Story

**As a** usuário com TDAH,
**I want** ver um relatório visual dos meus padrões de energia, humor e produtividade,
**so that** eu me conheça melhor e entenda o que funciona pra mim.

## Acceptance Criteria

1. Mapa de energia por horário (quando estou melhor/pior)
2. Padrão de humor por dia da semana
3. Taxa de conclusão de tarefas
4. Gatilhos detectados automaticamente
5. Recomendações personalizadas baseadas nos dados
6. Localizado em Revisões > Relatório Pessoal
7. Atualiza conforme mais dados são coletados

### Cálculo de Padrões

```typescript
interface PersonalReport {
  period: { start: string; end: string };

  energyMap: {
    hourly: { hour: number; avgEnergy: number; sampleSize: number }[];
    peakHours: number[];
    lowHours: number[];
  };

  moodPattern: {
    byDayOfWeek: { day: number; avgMood: number }[];
    bestDays: number[];
    worstDays: number[];
  };

  productivity: {
    completionRate: number;
    previousPeriodRate: number;
    byPriority: { high: number; medium: number; low: number };
    avgTasksPerDay: number;
  };

  detectedTriggers: {
    trigger: string;
    confidence: number;  // 0-1
    evidence: string;    // "Modo crise 80% das vezes após 22h"
  }[];

  recommendations: {
    text: string;
    category: 'schedule' | 'tasks' | 'breaks' | 'social';
    priority: number;
  }[];
}
```

### Algoritmo de Detecção de Gatilhos

```typescript
function detectTriggers(data: UserData): Trigger[] {
  const triggers: Trigger[] = [];

  // Correlação horário → energia baixa
  const lowEnergyHours = data.checkins
    .filter(c => c.energy <= 2)
    .map(c => new Date(c.timestamp).getHours());

  const hourCounts = countByHour(lowEnergyHours);
  const peakLowHour = maxBy(hourCounts, 'count');

  if (peakLowHour.count >= 5) {
    triggers.push({
      trigger: `Energia baixa às ${peakLowHour.hour}h`,
      confidence: peakLowHour.count / data.checkins.length,
      evidence: `${peakLowHour.count} ocorrências nos últimos 30 dias`
    });
  }

  return triggers;
}
```
