# Story 1.3: Tela de Questionário de Descoberta

## Status
Draft

## Story

**As a** usuário,
**I want** responder perguntas simples sobre como meu cérebro funciona,
**so that** o sistema entenda meu perfil.

## Acceptance Criteria

1. Interface de questionário com uma pergunta por tela
2. Navegação por deslizar ou toque
3. Progresso visual mostrando etapa atual
4. Perguntas cobrem: dificuldades, energia/horários, estilo de execução, sobrecarga
5. Tempo total de 5-7 minutos
6. Linguagem humana e não-clínica em todas as perguntas
7. Possibilidade de pausar e continuar depois

## Dev Notes

### Schema de Pergunta
```typescript
// packages/shared/src/types/discovery.ts
interface DiscoveryQuestion {
  id: string;
  category: 'difficulties' | 'energy' | 'execution' | 'overload';
  question: string;
  helperText?: string;
  options: {
    id: string;
    emoji?: string;
    label: string;
    description?: string;
    value: number; // para scoring
  }[];
}

interface DiscoveryAnswer {
  questionId: string;
  optionId: string;
  answeredAt: Date;
}
```
