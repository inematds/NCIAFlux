# Story 7.9: Gamificação Básica TDAH-Friendly

## Status
Draft

## Story

**As a** usuário com TDAH,
**I want** receber celebrações e feedback positivo imediato quando completo tarefas,
**so that** eu tenha dopamina imediata e motivação para continuar.

## Acceptance Criteria

1. Animação de celebração ao completar tarefa
2. Mensagens de encorajamento variadas
3. Sem punição por falhas (nunca zerar progresso)
4. Streaks gentis (mostram pausa, não perda)
5. Configurável: intensidade de gamificação
6. Funciona mesmo no plano Free (básico)

## Tasks / Subtasks

- [ ] **Celebrações** (AC: 1, 2)
  - [ ] Animação confetti/sparkle ao completar tarefa
  - [ ] Banco de ~30 mensagens de encorajamento
  - [ ] Variação baseada no contexto (primeira do dia, última, difícil)
  - [ ] Som opcional de celebração

- [ ] **Streaks Gentis** (AC: 3, 4)
  - [ ] Contador de dias ativos
  - [ ] "Pausa" ao invés de "perdeu"
  - [ ] Mensagem: "Descansou 1 dia. Sem problemas! 🌟"
  - [ ] Nunca zerar para 0

- [ ] **"Vitória do Dia"** (AC: 1)
  - [ ] Card no dashboard
  - [ ] Destaca maior conquista do dia
  - [ ] Mensagem motivacional personalizada

- [ ] **Configurações** (AC: 5)
  - [ ] Toggle gamificação ON/OFF
  - [ ] Intensidade: Sutil / Moderado / Completo
  - [ ] Sons ON/OFF
  - [ ] Animações ON/OFF

### Streak Gentil

```typescript
interface GentleStreak {
  currentStreak: number;      // Dias ativos
  longestStreak: number;      // Recorde
  lastActiveDate: string;
  pauseDays: number;          // Dias de "pausa" (não penaliza)

  // Nunca mostramos "perdeu streak"
  // Mostramos "pausou X dias" e continuamos
}

function getStreakMessage(streak: GentleStreak): string {
  const daysSinceActive = getDaysSince(streak.lastActiveDate);

  if (daysSinceActive === 0) {
    return `🔥 ${streak.currentStreak} dias ativos!`;
  } else if (daysSinceActive <= 3) {
    return `😴 Descansando há ${daysSinceActive} dia(s). Sem problemas!`;
  } else {
    return `👋 Sentimos sua falta! Que tal voltar hoje?`;
  }
}
```

### Animação de Celebração

```typescript
// Usando canvas-confetti ou similar
import confetti from 'canvas-confetti';

function celebrateTaskComplete(intensity: 'subtle' | 'moderate' | 'full') {
  if (intensity === 'subtle') {
    // Apenas sparkle sutil
    showSparkle();
  } else if (intensity === 'moderate') {
    // Confetti moderado
    confetti({ particleCount: 50, spread: 60 });
  } else {
    // Confetti completo + som
    confetti({ particleCount: 100, spread: 100 });
    playSound('celebration');
  }
}
```

## Princípios TDAH

1. **Dopamina imediata**: Feedback em <500ms após ação
2. **Sem punição**: Nunca mostrar "você falhou" ou "perdeu"
3. **Celebrar pequeno**: Cada tarefa merece reconhecimento
4. **Variedade**: Mensagens diferentes para não ficar repetitivo
5. **Opt-out fácil**: Quem não quer, desliga facilmente
