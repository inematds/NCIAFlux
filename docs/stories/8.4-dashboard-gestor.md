# Story 8.4: Dashboard do Gestor

## Status
Draft

## Story

**As a** gestor de time,
**I want** ver uma visão geral do time em um dashboard,
**so that** eu possa acompanhar a saúde do grupo.

## Acceptance Criteria

1. Métricas agregadas do time (anonimizadas)
2. Gráfico de tendência de estabilidade
3. Alertas de membros precisando apoio
4. Atividade recente do time
5. Não expõe dados individuais sensíveis
6. Customizável por gestor

## Tasks / Subtasks

- [ ] **Métricas Agregadas**
  - [ ] Média de estabilidade do time
  - [ ] Taxa de check-ins
  - [ ] Tarefas completadas (total)
  - [ ] Membros ativos

- [ ] **Visualizações**
  - [ ] Gráfico de linha temporal
  - [ ] Cards de KPIs
  - [ ] Lista de atividade

- [ ] **Alertas Inteligentes**
  - [ ] Membro inativo há X dias
  - [ ] Queda de estabilidade coletiva
  - [ ] Parabéns por conquistas

- [ ] **Privacidade**
  - [ ] Dados sempre agregados
  - [ ] Mínimo 3 membros para mostrar
  - [ ] Opt-out de métricas

## Dev Notes

```typescript
interface TeamDashboard {
  teamId: string;
  period: 'week' | 'month';
  metrics: {
    avgStability: number;
    checkInRate: number;
    activeMembers: number;
    totalMembers: number;
    tasksCompletedTotal: number;
    challengesActive: number;
  };
  trends: {
    date: string;
    avgStability: number;
    activeCount: number;
  }[];
  alerts: TeamAlert[];
  recentActivity: ActivityItem[];
}

interface TeamAlert {
  type: 'inactive_member' | 'stability_drop' | 'celebration';
  message: string;
  actionUrl?: string;
  priority: 'low' | 'medium' | 'high';
}
```

```sql
-- View para métricas agregadas (preserva privacidade)
CREATE VIEW team_aggregated_metrics AS
SELECT
  team_id,
  COUNT(DISTINCT user_id) as total_members,
  COUNT(DISTINCT user_id) FILTER (WHERE last_active > NOW() - INTERVAL '7 days') as active_members,
  AVG(stability_score) as avg_stability,
  -- Só mostra se tem >= 3 membros
  CASE WHEN COUNT(*) >= 3 THEN AVG(stability_score) ELSE NULL END as safe_avg_stability
FROM team_member_stats
GROUP BY team_id;
```
