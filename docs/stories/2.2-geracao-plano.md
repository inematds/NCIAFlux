# Story 2.2: Geração de Plano Diário

## Status
Draft

## Story

**As a** usuário do plano Básico,
**I want** receber um plano diário baseado no meu perfil,
**so that** tenha clareza do que fazer sem sobrecarga.

## Acceptance Criteria

1. Sistema gera plano com: 1 prioridade, máximo 2 tarefas leves, 1 bloco de foco
2. Plano considera perfil do usuário (energia, horários preferidos)
3. Plano é gerado automaticamente a cada dia
4. Usuário pode ajustar/personalizar plano manualmente
5. Técnicas sugeridas são no máximo 3 e relevantes ao perfil

## Dev Notes

### Tabela daily_plans
```sql
CREATE TABLE daily_plans (
    id UUID PRIMARY KEY DEFAULT uuid_generate_v4(),
    user_id UUID REFERENCES users(id) ON DELETE CASCADE NOT NULL,
    date DATE NOT NULL,
    priority JSONB, -- {title, description, status, estimated_minutes}
    tasks JSONB[] DEFAULT '{}', -- array de tasks leves
    focus_block JSONB, -- {start_time, duration_minutes, technique}
    suggested_techniques TEXT[] DEFAULT '{}',
    energy_level INTEGER CHECK (energy_level BETWEEN 1 AND 5),
    is_crisis_mode BOOLEAN DEFAULT FALSE,
    is_auto_generated BOOLEAN DEFAULT TRUE,
    created_at TIMESTAMPTZ DEFAULT NOW(),
    updated_at TIMESTAMPTZ DEFAULT NOW(),
    UNIQUE(user_id, date)
);

CREATE INDEX idx_daily_plans_user_date ON daily_plans(user_id, date DESC);
```

### Algoritmo de Geração
```typescript
interface GeneratePlanInput {
  userId: string;
  date: Date;
}

async function generateDailyPlan(input: GeneratePlanInput): Promise<DailyPlan> {
  const profile = await getProfile(input.userId);

  // Determinar horário de foco baseado em energia
  const focusTime = determineFocusTime(profile.energy_pattern);

  // Selecionar técnicas
  const techniques = selectTechniques(profile.preferred_techniques, 3);

  // Criar plano
  return {
    user_id: input.userId,
    date: input.date,
    priority: null, // usuário define
    tasks: [],
    focus_block: {
      start_time: focusTime,
      duration_minutes: 25, // pomodoro padrão
      technique: techniques[0],
    },
    suggested_techniques: techniques,
    is_auto_generated: true,
  };
}
```
