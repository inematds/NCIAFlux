# Story 7.12: Chat 1:1 (Accountability Partner)

## Status
Draft

## Story

**As a** usuário Plus,
**I want** conversar com outro usuário como parceiro de accountability,
**so that** tenhamos apoio mútuo na jornada.

## Acceptance Criteria

1. Adicionar parceiro por código/username
2. Chat em tempo real
3. Ver status do parceiro (online/offline)
4. Compartilhar conquistas automaticamente
5. Notificações de mensagens
6. Bloquear/remover parceiro

## Tasks / Subtasks

- [ ] **Sistema de Conexão**
  - [ ] Gerar código de parceria
  - [ ] Aceitar/recusar convite
  - [ ] Limite: 1-3 parceiros (por plano)

- [ ] **Chat Real-time**
  - [ ] Supabase Realtime
  - [ ] Mensagens de texto
  - [ ] Indicador de digitando
  - [ ] Lido/não lido

- [ ] **Compartilhamento**
  - [ ] Opt-in para compartilhar conquistas
  - [ ] "João completou 5 tarefas hoje!"
  - [ ] Celebrar junto

- [ ] **Privacidade**
  - [ ] Nunca compartilha humor/energia
  - [ ] Só conquistas e marcos
  - [ ] Bloquear a qualquer momento

## Dev Notes

```sql
CREATE TABLE partnerships (
  id UUID PRIMARY KEY,
  user_a UUID REFERENCES auth.users,
  user_b UUID REFERENCES auth.users,
  status TEXT DEFAULT 'pending', -- pending, active, blocked
  created_at TIMESTAMPTZ DEFAULT NOW()
);

CREATE TABLE direct_messages (
  id UUID PRIMARY KEY,
  partnership_id UUID REFERENCES partnerships,
  sender_id UUID REFERENCES auth.users,
  content TEXT,
  read_at TIMESTAMPTZ,
  created_at TIMESTAMPTZ DEFAULT NOW()
);
```
