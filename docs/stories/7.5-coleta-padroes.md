# Story 7.5: Coleta de Padrões

## Status
Draft

## Story

**As a** sistema Mentes Brilhantes,
**I want** coletar dados de uso do usuário de forma passiva,
**so that** possa detectar padrões e adaptar a experiência.

## Acceptance Criteria

1. Coletar horários de uso do app
2. Coletar taxa de conclusão de tarefas
3. Coletar frequência de modo crise
4. Coletar padrões de check-in (humor, energia)
5. Armazenar dados localmente e sincronizar com cloud
6. Respeitar privacidade (dados não saem sem consentimento)
