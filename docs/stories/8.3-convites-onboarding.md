# Story 8.3: Convites e Onboarding de Time

## Status
Draft

## Story

**As a** admin de time,
**I want** convidar pessoas para o time,
**so that** possamos crescer a equipe de forma controlada.

## Acceptance Criteria

1. Gerar link de convite único
2. Convite por email
3. Convite expira após X dias
4. Limite de convites pendentes
5. Onboarding guiado para novos membros
6. Aprovar/rejeitar solicitações

## Dev Notes

```sql
CREATE TABLE team_invitations (
  id UUID PRIMARY KEY DEFAULT uuid_generate_v4(),
  team_id UUID REFERENCES teams NOT NULL,
  code TEXT UNIQUE NOT NULL,
  email TEXT, -- Opcional, pode ser link genérico
  invited_by UUID REFERENCES auth.users NOT NULL,
  role TeamRole DEFAULT 'member',
  expires_at TIMESTAMPTZ NOT NULL,
  used_at TIMESTAMPTZ,
  used_by UUID REFERENCES auth.users,
  created_at TIMESTAMPTZ DEFAULT NOW()
);

CREATE TABLE team_join_requests (
  id UUID PRIMARY KEY DEFAULT uuid_generate_v4(),
  team_id UUID REFERENCES teams NOT NULL,
  user_id UUID REFERENCES auth.users NOT NULL,
  status TEXT DEFAULT 'pending', -- pending, approved, rejected
  reviewed_by UUID REFERENCES auth.users,
  reviewed_at TIMESTAMPTZ,
  created_at TIMESTAMPTZ DEFAULT NOW(),
  UNIQUE(team_id, user_id)
);
```
