# Story 8.8: Rankings Amigáveis

## Status
Draft

## Story

**As a** membro de time,
**I want** ver rankings que celebram sem criar pressão,
**so that** eu tenha motivação saudável.

## Acceptance Criteria

1. Rankings opt-in (não obrigatório)
2. Foco em crescimento, não posição
3. Múltiplas categorias (não só produtividade)
4. Anonimização opcional
5. Nunca mostra "últimos lugares"
6. Celebra melhoria, não só liderança

## Tasks / Subtasks

- [ ] **Sistema de Rankings**
  - [ ] Opt-in por usuário
  - [ ] Categorias diversas
  - [ ] Período (semanal/mensal)

- [ ] **Categorias**
  - [ ] Maior crescimento (delta)
  - [ ] Streak mais longo
  - [ ] Mais conquistas desbloqueadas
  - [ ] Mais desafios participados
  - [ ] NÃO: Mais tarefas (evita burnout)

- [ ] **Visualização**
  - [ ] Top 3 com destaque
  - [ ] Mostrar apenas quem está acima
  - [ ] "Você subiu X posições!"

- [ ] **Proteções**
  - [ ] Nunca mostra últimos
  - [ ] Opção de anonimato
  - [ ] Foco no progresso pessoal
