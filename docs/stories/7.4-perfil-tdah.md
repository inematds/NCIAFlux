# Story 7.4: Perfil TDAH e Ferramentas de Avaliação

## Status
Draft

## Story

**As a** usuário com TDAH,
**I want** responder um questionário sobre meu subtipo e poder editar/refazer quando quiser,
**so that** o app me entenda melhor e adapte a experiência.

## Acceptance Criteria

1. Questionário de subtipo TDAH (hiperativo/desatento/combinado)
2. Questionário de gatilhos conhecidos
3. Resultado salvo no perfil do usuário
4. Botão "Refazer avaliação" sempre disponível
5. Botão "Editar manualmente" para ajuste direto
6. Histórico de mudanças do perfil
7. Integração com cronotipo existente

## Tasks / Subtasks

- [ ] **Questionário TDAH** (AC: 1)
  - [ ] Perguntas baseadas em DSM-5 simplificadas
  - [ ] ~10 perguntas com escala Likert
  - [ ] Algoritmo de pontuação por subtipo
  - [ ] Resultado: hiperativo / desatento / combinado / inconclusivo

- [ ] **Questionário de Gatilhos** (AC: 2)
  - [ ] Lista de gatilhos comuns (checklist)
  - [ ] Campo para adicionar gatilhos personalizados
  - [ ] Categorias: tempo, ambiente, social, cognitivo

- [ ] **UI do Perfil** (AC: 3, 4, 5, 7)
  - [ ] Seção no Configurações > Perfil
  - [ ] Exibir cronotipo atual
  - [ ] Exibir subtipo TDAH atual
  - [ ] Exibir gatilhos salvos
  - [ ] Botão [Refazer] para cada questionário
  - [ ] Botão [Editar] para ajuste manual

- [ ] **Edição Manual** (AC: 5)
  - [ ] Dropdown para selecionar subtipo diretamente
  - [ ] Lista editável de gatilhos
  - [ ] Sliders para preferências (ex: max tarefas visíveis)

- [ ] **Histórico** (AC: 6)
  - [ ] Salvar data de cada avaliação
  - [ ] Motivo da mudança (opcional)
  - [ ] Visualizar evolução ao longo do tempo
