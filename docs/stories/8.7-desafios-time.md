# Story 8.7: Desafios de Time

## Status
Draft

## Story

**As a** membro de time,
**I want** participar de desafios coletivos,
**so that** tenhamos motivação compartilhada.

## Acceptance Criteria

1. Gestor cria desafios para o time
2. Desafios cooperativos (todos contribuem)
3. Meta coletiva (não individual)
4. Celebração quando time atinge meta
5. Desafios NUNCA punem
6. Participação opcional

## Tasks / Subtasks

- [ ] **Criação de Desafio**
  - [ ] Nome e descrição
  - [ ] Tipo de métrica (tarefas, check-ins, etc)
  - [ ] Meta coletiva
  - [ ] Duração (início/fim)

- [ ] **Participação**
  - [ ] Opt-in/opt-out
  - [ ] Contribuição individual (privada)
  - [ ] Progresso coletivo (público)

- [ ] **Progresso**
  - [ ] Barra de progresso do time
  - [ ] Atualizações em tempo real
  - [ ] Mensagens de incentivo

- [ ] **Celebração**
  - [ ] Notificação ao completar
  - [ ] Badge de time
  - [ ] XP bonus para todos

```typescript
interface TeamChallenge {
  id: string;
  teamId: string;
  name: string;
  description: string;
  metric: 'tasks' | 'checkins' | 'focus_minutes' | 'streak_days';
  targetValue: number;
  currentValue: number;
  startDate: string;
  endDate: string;
  status: 'active' | 'completed' | 'expired';
  participants: string[]; // userIds que optaram participar
  reward: {
    xpBonus: number;
    badge?: string;
  };
  createdBy: string;
}

interface ChallengeContribution {
  challengeId: string;
  date: string;
  // Contribuições individuais são privadas
  contributions: {
    odim_id: string; // hash para privacidade
    value: number;
  }[];
  totalValue: number;
}
```

```sql
CREATE TABLE team_challenges (
  id UUID PRIMARY KEY DEFAULT uuid_generate_v4(),
  team_id UUID REFERENCES teams NOT NULL,
  name TEXT NOT NULL,
  description TEXT,
  metric TEXT NOT NULL,
  target_value INT NOT NULL,
  current_value INT DEFAULT 0,
  start_date DATE NOT NULL,
  end_date DATE NOT NULL,
  status TEXT DEFAULT 'active',
  reward JSONB DEFAULT '{"xpBonus": 50}',
  created_by UUID REFERENCES auth.users NOT NULL,
  created_at TIMESTAMPTZ DEFAULT NOW()
);

CREATE TABLE challenge_participants (
  challenge_id UUID REFERENCES team_challenges,
  user_id UUID REFERENCES auth.users,
  joined_at TIMESTAMPTZ DEFAULT NOW(),
  contribution INT DEFAULT 0,
  PRIMARY KEY (challenge_id, user_id)
);
```
