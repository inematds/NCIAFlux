# Story 7.13: Chat em Grupo (Comunidades)

## Status
Draft

## Story

**As a** usuário Plus,
**I want** participar de comunidades de apoio,
**so that** eu possa trocar experiências com pessoas similares.

## Acceptance Criteria

1. Criar/entrar em comunidades
2. Chat em grupo em tempo real
3. Moderação básica (admin pode remover)
4. Tópicos/canais dentro da comunidade
5. Limite de membros por comunidade
6. Regras de comunidade visíveis

## Dev Notes

```sql
CREATE TABLE communities (
  id UUID PRIMARY KEY,
  name TEXT NOT NULL,
  description TEXT,
  rules TEXT,
  owner_id UUID REFERENCES auth.users,
  is_public BOOLEAN DEFAULT true,
  max_members INT DEFAULT 50,
  created_at TIMESTAMPTZ DEFAULT NOW()
);

CREATE TABLE community_members (
  community_id UUID REFERENCES communities,
  user_id UUID REFERENCES auth.users,
  role TEXT DEFAULT 'member', -- admin, moderator, member
  joined_at TIMESTAMPTZ DEFAULT NOW(),
  PRIMARY KEY (community_id, user_id)
);
```
