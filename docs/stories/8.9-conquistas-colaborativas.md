# Story 8.9: Conquistas Colaborativas

## Status
Draft

## Story

**As a** membro de time,
**I want** desbloquear conquistas junto com meu time,
**so that** celebremos vitórias coletivas.

## Acceptance Criteria

1. Conquistas que só se desbloqueiam em grupo
2. Todos do time recebem ao desbloquear
3. Galeria de conquistas do time
4. Histórico de quando foram desbloqueadas
5. Mensagens de celebração coletiva
6. Conquistas raras para times dedicados

## Tasks / Subtasks

- [ ] **Sistema de Conquistas**
  - [ ] Definir ~15 conquistas de time
  - [ ] Critérios coletivos
  - [ ] Níveis de raridade

- [ ] **Desbloqueio**
  - [ ] Detectar quando critério atinge
  - [ ] Notificar todos membros
  - [ ] Animação de celebração

- [ ] **Galeria do Time**
  - [ ] Lista de conquistas
  - [ ] Data de desbloqueio
  - [ ] Quem estava no time

- [ ] **Conquistas Especiais**
  - [ ] Primeiro mês ativo
  - [ ] 100% participação em desafio
  - [ ] Time ajudou membro em crise

```sql
CREATE TABLE team_achievements (
  id UUID PRIMARY KEY DEFAULT uuid_generate_v4(),
  team_id UUID REFERENCES teams NOT NULL,
  achievement_id TEXT NOT NULL,
  unlocked_at TIMESTAMPTZ DEFAULT NOW(),
  members_at_time UUID[] NOT NULL,
  UNIQUE(team_id, achievement_id)
);

-- Trigger para notificar todos
CREATE OR REPLACE FUNCTION notify_team_achievement()
RETURNS TRIGGER AS $$
BEGIN
  -- Inserir notificação para cada membro
  INSERT INTO notifications (user_id, type, data)
  SELECT
    unnest(NEW.members_at_time),
    'team_achievement',
    jsonb_build_object(
      'teamId', NEW.team_id,
      'achievementId', NEW.achievement_id
    );
  RETURN NEW;
END;
$$ LANGUAGE plpgsql;

CREATE TRIGGER on_team_achievement
AFTER INSERT ON team_achievements
FOR EACH ROW EXECUTE FUNCTION notify_team_achievement();
```
