# Story 7.2: Sync Offline-First

## Status
Draft

## Story

**As a** usuário Plus,
**I want** que meus dados sincronizem automaticamente quando estiver online,
**so that** eu possa usar o app offline e nunca perder dados.

## Acceptance Criteria

1. App funciona 100% offline com dados locais
2. Quando online, sincroniza automaticamente em background
3. Conflitos resolvidos automaticamente (last-write-wins ou merge)
4. Indicador visual de status de sync (sincronizado/sincronizando/offline)
5. Retry automático em caso de falha de rede
6. Multi-device: dados aparecem em todos os dispositivos logados

## Dev Notes

### Schema Principal

```sql
CREATE TABLE tasks (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  user_id UUID REFERENCES auth.users NOT NULL,
  title TEXT NOT NULL,
  description TEXT,
  status TEXT DEFAULT 'pending',
  priority TEXT DEFAULT 'medium',
  due_date DATE,
  project_id UUID,
  created_at TIMESTAMPTZ DEFAULT NOW(),
  updated_at TIMESTAMPTZ DEFAULT NOW(),
  deleted_at TIMESTAMPTZ  -- soft delete
);

-- Index para sync
CREATE INDEX tasks_user_updated ON tasks(user_id, updated_at);

-- RLS
ALTER TABLE tasks ENABLE ROW LEVEL SECURITY;
CREATE POLICY "Users own tasks" ON tasks
  FOR ALL USING (auth.uid() = user_id);
```

### Sync Strategy

```typescript
interface SyncState {
  lastSyncAt: string | null;
  pendingOperations: Operation[];
  status: 'synced' | 'syncing' | 'offline' | 'error';
}

// Push: local → cloud
async function pushChanges(table: string, since: string) {
  const localChanges = getLocalChangesSince(table, since);
  await supabase.from(table).upsert(localChanges);
}

// Pull: cloud → local
async function pullChanges(table: string, since: string) {
  const { data } = await supabase
    .from(table)
    .select('*')
    .gt('updated_at', since);
  mergeToLocal(table, data);
}
```

### Realtime Subscription

```typescript
supabase
  .channel('tasks')
  .on('postgres_changes',
    { event: '*', schema: 'public', table: 'tasks' },
    (payload) => {
      if (payload.new.user_id === currentUser.id) {
        updateLocalStore(payload);
      }
    }
  )
  .subscribe();
```
