# Story 7.3: Importação em Massa

## Status
Draft

## Story

**As a** usuário,
**I want** importar meus dados de outras ferramentas (CSV, ICS, JSON),
**so that** eu possa migrar para o Mentes Brilhantes sem perder histórico.

## Acceptance Criteria

1. Importar tarefas via CSV (titulo, descricao, prioridade, data)
2. Importar eventos via ICS (padrão iCalendar)
3. Importar notas via CSV (titulo, conteudo, pasta)
4. Importar brain dump via CSV ou TXT
5. Restaurar backup completo via JSON
6. Prévia dos dados antes de confirmar importação
7. Tratamento de erros com feedback claro

## Tasks / Subtasks

- [ ] **UI de Importação** (AC: 6, 7)
  - [ ] Tela Configurações > Dados > Importar
  - [ ] Seletor de tipo de importação
  - [ ] Upload de arquivo
  - [ ] Tela de prévia com validação
  - [ ] Feedback de sucesso/erro

- [ ] **Parser CSV** (AC: 1, 3, 4)
  - [ ] Função genérica de parse CSV
  - [ ] Mapeamento de colunas para schema
  - [ ] Validação de campos obrigatórios
  - [ ] Tratamento de encoding (UTF-8)

- [ ] **Parser ICS** (AC: 2)
  - [ ] Parse de formato iCalendar
  - [ ] Extrair VEVENT (título, data, hora, descrição)
  - [ ] Converter timezone
  - [ ] Suporte a eventos recorrentes (básico)

- [ ] **Importação Brain Dump** (AC: 4)
  - [ ] CSV: conteudo, categoria (opcional)
  - [ ] TXT: um item por linha
  - [ ] Auto-categorização opcional

- [ ] **Restauração de Backup** (AC: 5)
  - [ ] Validar estrutura do JSON
  - [ ] Confirmar substituição de dados
  - [ ] Importar todas as entidades

## Dev Notes

### Formatos Esperados

**CSV Tarefas:**
```csv
titulo,descricao,prioridade,data,projeto
"Fazer relatório","Relatório mensal de vendas",high,2026-02-01,Trabalho
"Comprar leite",,low,,Pessoal
```

**CSV Notas:**
```csv
titulo,conteudo,pasta
"Ideias App","Lista de features para implementar",Projetos
"Receita bolo","Ingredientes: farinha, ovos...",Pessoal
```

**CSV/TXT Brain Dump:**
```csv
conteudo,categoria
"Ligar pro dentista",urgente
"Ideia: curso de Python",ideias
```

```txt
Ligar pro dentista
Ideia: curso de Python
Comprar presente Maria
```

**ICS (iCalendar):**
```ics
BEGIN:VCALENDAR
VERSION:2.0
BEGIN:VEVENT
DTSTART:20260201T100000
DTEND:20260201T110000
SUMMARY:Reunião de equipe
DESCRIPTION:Alinhamento semanal
END:VEVENT
END:VCALENDAR
```

### Fluxo de Importação

```
Selecionar tipo → Upload arquivo → Prévia

                               │         │

                         [Importar]   [Ver erros]
                               │
                               ▼
                         "47 itens importados!"
```
