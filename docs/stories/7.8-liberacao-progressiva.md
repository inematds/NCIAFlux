# Story 7.8: Liberação Progressiva de Features

## Status
Draft

## Story

**As a** usuário com TDAH,
**I want** que features avançadas sejam liberadas conforme minha estabilidade,
**so that** eu não fique sobrecarregado com muitas opções de uma vez.

## Acceptance Criteria

1. Sistema de níveis de features (1-5)
2. Liberação automática baseada em estabilidade
3. Notificação quando nova feature é liberada
4. Visualização de features bloqueadas e requisitos
5. Override por gestor/terapeuta
6. Override manual (com aviso)

## Tasks / Subtasks

- [ ] **Sistema de Níveis**
  - [ ] Nível 1: Brain Dump, Tarefas básicas, Modo Crise, Chat IA
  - [ ] Nível 2: Planner, Rotinas, Projetos, Modo Foco
  - [ ] Nível 3: Todas tarefas, Relatório, Revisão, Calendário
  - [ ] Nível 4: Multi-device, Chat 1:1, Comunidade
  - [ ] Nível 5: Todas features

- [ ] **Lógica de Liberação**
  - [ ] Nível 2: Perfil completo
  - [ ] Nível 3: Estabilidade > 50%
  - [ ] Nível 4: Estabilidade > 70% + Plano Plus
  - [ ] Nível 5: Gestor/Terapeuta libera

- [ ] **UI de Features**
  - [ ] Ícone de cadeado em features bloqueadas
  - [ ] Tooltip com requisito
  - [ ] Celebração ao desbloquear

- [ ] **Overrides**
  - [ ] Gestor pode liberar para membro
  - [ ] Terapeuta pode liberar
  - [ ] Usuário pode forçar (com log)
