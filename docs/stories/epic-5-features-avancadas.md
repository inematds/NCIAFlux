# Epic 5: Features Avançadas

**Objetivo:** Implementar funcionalidades do plano Avançado: relatórios avançados, biblioteca de conteúdo educativo e comunidade.

**Valor de Negócio:** Diferenciação premium com valor real para usuários engajados.

**Dependências:** Epic 1-4

**Stories:**
- [5.1 Biblioteca de Relatórios](./5.1-biblioteca-relatorios.md)
- [5.2 Chat por Voz](./5.2-chat-voz.md)
- [5.3 Sistema de Conteúdo Educativo](./5.3-conteudo-educativo.md)
- [5.4 Comunidade e Times](./5.4-comunidade.md)

**Definition of Done do Epic:**
- [ ] Relatórios avançados acessíveis
- [ ] Chat por voz funcional
- [ ] CMS de conteúdo educativo operacional
- [ ] Comunidade com moderação básica

---

## Story 5.1: Biblioteca de Relatórios (Resumo)

**As a** usuário do plano Avançado,
**I want** acessar diferentes tipos de relatórios,
**so that** entenda meus padrões quando quiser.

**Key Tasks:**
- Tela de biblioteca com cards de relatórios
- Relatório de padrões de energia
- Relatório de técnicas eficazes
- Relatório de tendências semanais/mensais
- Visualizações visuais, não tabelas

---

## Story 5.2: Chat por Voz (Resumo)

**As a** usuário do plano Avançado,
**I want** interagir com o chat usando voz,
**so that** seja ainda mais fácil fazer check-ins.

**Key Tasks:**
- Botão de voz no chat
- Speech-to-text (nativo ou API)
- Text-to-speech para respostas
- Funcionar offline com sync

---

## Story 5.3: Sistema de Conteúdo Educativo (Resumo)

**As a** usuário do plano Avançado,
**I want** acessar pílulas educativas sobre TDAH,
**so that** aprenda técnicas e entenda melhor meu cérebro.

**Key Tasks:**
- Setup Strapi CMS
- Tela de pílulas educativas
- Controle de liberação por gestão
- Progresso de leitura salvo

---

## Story 5.4: Comunidade e Times (Resumo)

**As a** usuário do plano Avançado,
**I want** participar de uma comunidade de apoio,
**so that** me sinta menos sozinho na jornada.

**Key Tasks:**
- Feature toggle pela gestão
- Feed de comunidade
- Times temáticos
- Sistema de moderação básico
- Denúncia de conteúdo
