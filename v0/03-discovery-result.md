# v0.dev Prompt - Resultado da Descoberta

## Prompt

```
Create a mobile results screen for "NCIAFlux" ADHD app. This screen shows the user's cognitive profile after completing the discovery questionnaire. It should feel like a warm revelation, not a clinical diagnosis.

**Design Philosophy:**
- Celebratory but calm
- Personal and empathetic
- Value delivered immediately
- No pressure to continue

**Color Palette:**
- Primary: #4A90A4
- Secondary: #E8A87C
- Accent/Success: #85C88A
- Background: #F9FAFB
- Surface: #FFFFFF
- Text Primary: #2D3748
- Text Secondary: #718096

**Screen Layout:**

1. **Header:**
   - Subtle celebration animation (confetti or gentle particles)
   - Title: "Seu Perfil" (small, secondary)

2. **Profile Card (Main Hero):**
   - Large card with soft gradient background (primary light to white)
   - User's profile summary in 1-2 sentences:
     "Você tem um cérebro criativo que funciona melhor em rajadas de energia, especialmente à noite. Estruturas rígidas te sufocam."
   - Typography: 20px, Nunito, semibold, dark text
   - Subtle brain/flow icon

3. **Insight Card:**
   - Quote-style design with left border accent (secondary color)
   - Label: "💡 Seu Insight Principal"
   - Insight text: "Você não é preguiçoso - seu cérebro precisa de interesse para ativar. Tarefas 'chatas' drenam você 10x mais que os outros."
   - Typography: 16px, italic feel, warm tone

4. **Suggestion Card:**
   - Icon: lightbulb or arrow
   - Label: "🎯 Próximo Passo Prático"
   - Suggestion: "Experimente fazer suas tarefas mais difíceis no horário que você naturalmente tem mais energia, mesmo que seja 'estranho'."
   - Small actionable feel

5. **Action Buttons (bottom):**
   - Primary: "Continuar com NCIAFlux" (full width, primary color)
   - Secondary row:
     - "Salvar Resultado" (outline button with download icon)
     - "Compartilhar" (outline button with share icon)
   - Ghost link: "Só quero ver, obrigado" (no pressure exit)

6. **Footer:**
   - Small text: "Seu resultado foi gerado com base em pesquisas sobre TDAH e neurodiversidade"

**Visual Details:**
- Cards with shadow-md, rounded-2xl (16px)
- Generous padding (24px)
- Smooth scroll if content exceeds screen
- Entry animation: cards fade in sequentially

**Mobile dimensions:** 390x844
```

## Notas
- Esta tela entrega o valor principal do app gratuito
- Deve fazer o usuário se sentir compreendido
- Botão de sair sem pressão é importante
