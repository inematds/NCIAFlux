# v0.dev Prompt - Tela de Boas-vindas

## Prompt

```
Create a mobile-first welcome screen for "NCIAFlux" - a calming ADHD support app. The design should feel warm, inviting, and NOT clinical.

**Brand:**
- App name: NCIAFlux (subtitle: NeuroFluxo)
- Tagline: "Autoconhecimento • Adaptação • Sem Pressão"
- Logo: Abstract flowing brain with colorful lines (use a placeholder)

**Color Palette:**
- Primary: #4A90A4 (teal blue)
- Secondary: #E8A87C (warm peach)
- Accent: #85C88A (soft green)
- Background: #F9FAFB (off-white)
- Text Primary: #2D3748
- Text Secondary: #718096

**Typography:**
- Headings: Nunito, bold
- Body: Inter, regular

**Screen Elements:**
1. Logo centered at top (with subtle animation feel)
2. Welcome message: "Entenda como seu cérebro funciona" (large, friendly)
3. Subtext: "Uma jornada de autoconhecimento em 5 minutos" (smaller, secondary color)
4. Primary button: "Começar Descoberta" (full width, primary color, rounded 12px)
5. Secondary text link: "Já tenho conta" (ghost style)
6. Bottom decorative illustration (abstract, calming waves or flow)

**Design Principles:**
- Generous whitespace
- Soft shadows (shadow-sm)
- Border radius: 12-16px on cards/buttons
- No harsh colors or urgent elements
- Touch targets minimum 44px height
- Calming, breathable layout

**Mobile dimensions:** 390x844 (iPhone 14 Pro)
```

## Notas
- Esta é a primeira tela que o usuário vê
- Deve transmitir calma e acolhimento
- Nenhuma pressão para cadastro
