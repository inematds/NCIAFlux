# v0.dev Prompt - Modo Crise

## Prompt

```
Create a mobile "Crisis Mode" screen for "NCIAFlux" ADHD app. This is an ultra-simplified interface for overwhelming days. It should feel like a warm hug, not a failure state.

**Design Philosophy:**
- MAXIMUM simplicity
- No judgment
- One thing only
- Calming, safe space feeling
- Easy to exit when ready

**Color Palette:**
- Background: Soft gradient from #E8EEF2 to #F5F7FA (calming blue-gray)
- Primary: #4A90A4
- Accent: #85C88A
- Text Primary: #2D3748
- Text Secondary: #718096
- Card: #FFFFFF with very subtle shadow

**Screen Layout:**

1. **Background:**
   - Soft, calming gradient (not flat)
   - Optional: very subtle animated particles or waves (slow, gentle)
   - Reduced visual complexity

2. **Header (minimal):**
   - Small cloud or shelter icon (centered)
   - "Modo Crise Ativo" (small text, muted)
   - No back button prominently visible

3. **Main Message (centered, hero):**
   - Large, warm text:
     "Está tudo bem."
   - Subtitle:
     "Hoje é dia de cuidar de você."
   - Typography: 28px heading, 18px subtitle
   - Generous line height
   - Centered, plenty of whitespace

4. **Single Task Card:**
   - Simple white card, centered
   - Very rounded (rounded-3xl)
   - Content:
     - Small label: "Sua única missão"
     - Task: "Beba um copo de água 💧"
     - OR: "Respire fundo 3 vezes 🌬️"
     - OR: "Apenas descanse 🛋️"
   - Optional checkbox (not required)
   - No time pressure, no metrics

5. **Supportive Message:**
   - Below the card
   - "Notificações pausadas até amanhã"
   - "Seu plano volta ao normal quando você quiser"
   - Small, muted text

6. **Exit Options (bottom, subtle):**
   - "Estou melhor, desativar modo crise"
   - Ghost button style, not prominent
   - Tap reveals confirmation

7. **No Bottom Navigation:**
   - Clean screen
   - Swipe or tap anywhere to access menu if needed
   - Focus on calm, not navigation

**Animation:**
- Entry: gentle fade in (400ms)
- Background: very slow, subtle movement
- Respect prefers-reduced-motion

**States:**
- Active: as described above
- Transitioning out: "Que bom que você está melhor 💚" message before returning

**Mobile dimensions:** 390x844
```

## Mensagens Alternativas
Variar a "única missão" para não ser repetitivo:
- "Beba água"
- "Respire fundo"
- "Descanse um pouco"
- "Dê uma volta curta"
- "Coma algo leve"
- "Apenas exista - isso é suficiente"
