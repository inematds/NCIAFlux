# v0.dev Prompt - Meu Perfil Cognitivo

## Prompt

```
Create a mobile cognitive profile screen for "NCIAFlux" ADHD app. This shows the user's brain profile generated from discovery, and how it has evolved over time. Make it feel personal and insightful, not clinical.

**Design Philosophy:**
- Personal, not medical
- Show growth, not judgment
- Insights over metrics
- Option to rediscover if things change

**Color Palette:**
- Primary: #4A90A4
- Secondary: #E8A87C
- Accent: #85C88A
- Background: #F9FAFB
- Surface: #FFFFFF
- Text Primary: #2D3748
- Text Secondary: #718096
- Chart colors: #4A90A4, #E8A87C, #85C88A, #9F7AEA

**Screen Layout:**

1. **Header:**
   - Back arrow
   - Title: "Seu Perfil"
   - Share icon (top right)

2. **Profile Summary Card (Hero):**
   - User name/avatar: "Ana"
   - Profile since: "Desde Janeiro 2026"
   - Main summary (2 sentences):
     "Cérebro criativo que funciona em rajadas de energia. Melhor desempenho à noite, prefere variedade a rotina."
   - Card: gradient border (primary to secondary), white inside

3. **Key Traits Section:**
   - "🧠 Características Principais"
   - Visual tags/chips:
     - "Criativo"
     - "Noturno"
     - "Rajadas de energia"
     - "Sensível a sobrecarga"
   - Chips with soft colors, rounded-full

4. **Energy Pattern Visualization:**
   - "⚡ Seu padrão de energia"
   - Simple line chart or bar chart showing energy by time of day:
     - Morning: Low (short bar, muted)
     - Afternoon: Medium (medium bar)
     - Evening: High (tall bar, accent color)
   - Clean, minimal chart design
   - Labels: "Manhã", "Tarde", "Noite"

5. **Effective Techniques:**
   - "✅ O que funciona para você"
   - List of techniques with effectiveness indicator:
     - "Pomodoro adaptado" ●●●○ (works well)
     - "Brain dump" ●●●● (works great)
     - "Starter step" ●●○○ (moderate)
   - Based on usage data

6. **Insights Section:**
   - "💡 Insights do seu uso"
   - Cards with discovered patterns:
     - "Você completa mais tarefas quando começa pequeno"
     - "Terças e quintas são seus dias mais produtivos"
     - "Blocos de 25min funcionam melhor que 45min"
   - Show only 2-3, expandable

7. **Profile Evolution:**
   - "📈 Evolução"
   - Small timeline or before/after:
     - "Quando começou: energia imprevisível"
     - "Agora: padrões mais claros, técnicas definidas"
   - Positive framing only

8. **Actions:**
   - "Refazer Descoberta" (outline button)
     - Subtitle: "Se sua rotina mudou muito"
   - "Compartilhar Perfil" (outline button)

9. **Last Updated:**
   - "Perfil atualizado: há 3 dias"
   - Small, bottom of screen

**Visual Details:**
- Soft, friendly data visualizations
- No hard numbers unless meaningful
- Progress shown as improvement, not scores
- Personal language ("seu", "você")

**Mobile dimensions:** 390x844
```

## Nota
- Este perfil evolui com o uso
- Não é estático como no resultado inicial
- Mostra aprendizados do sistema
